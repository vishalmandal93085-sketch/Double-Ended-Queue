Deque Implementation Using Linked List in C
📌 Overview

This project demonstrates the implementation of a Deque (Double Ended Queue) using a singly linked list in the C programming language.
A deque allows insertion and deletion of elements from both the front and the rear.

✨ Features

Insert elements at the front

Insert elements at the rear

Delete elements from the front

Delete elements from the rear

Display all elements in the deque

Dynamic memory allocation using malloc

🧠 Data Structure Used

Linked List

Each node contains:

data → integer value

next → pointer to the next node

Two pointers are maintained:

front → points to the first node

rear → points to the last node

⚙️ Functions Included

insertFront(int value) – Inserts an element at the front

insertRear(int value) – Inserts an element at the rear

deleteFront() – Deletes an element from the front

deleteRear() – Deletes an element from the rear

display() – Displays all elements of the deque
